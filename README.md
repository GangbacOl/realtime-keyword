# 실시간 검색어(크롬 확장 프로그램)

최근에 중단된 여러 포털사이트들의 실시간 검색어 서비스를 크롬 확장 프로그램으로 제공해보고자 제작하게 된 프로젝트입니다.

실시간 검색어 정보들은 [Google Trends](https://trends.google.com/)에서 제공되는 RSS 피드를 기반으로 하고 있으며, 1시간을 주기로 업데이트 됩니다.
